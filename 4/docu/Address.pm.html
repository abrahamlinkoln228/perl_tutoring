  Address manage functions

address_info($id) - Address register full address info

  Arguments:
    $id  - Location id

  Returns:
    Address object
      DISTRICT_ID
      CITY
      ADDRESS_DISTRICT
      ADDRESS_STREET
      ADDRESS_BUILD
      STREET_ID
      ZIP
      COORDX

address_list($attr) - returns list of all builds in DB

  Arguments:
    $attr - hash_ref, reserved for future

  Returns:
    Address_list
      DISTRICT_ID
      DISTRICT_NAME
      STREET_ID
      STREET_NAME
      STREET_DISTRICT_ID
      BUILD_ID
      BUILD_NAME
      BUILD_STREET_ID

address_parentness($in_array_func, $attr) - Build an parentness tree for districts->streets->builds

  We need to pass here an $in_array function to avoid using Base.pm

  Arguments:
    $in_array_func - refference for a in_array function
    $attr - hash_ref
      STREETS - boolean (do not make builds)



  Returns:
    $list - DB list of builds
    $parentness_hash - districts->streets->builds ierarchy. You can read more doc in HTML::build_parentness_tree()
