new($db, $admin, $CONF)

  Arguments:
    $db    - ref to DB
    $admin - current Web session admin
    $CONF  - ref to %conf

  Returns:
    object

execute($cmd, $attr) - Execute command in remote console

  Arguments:
    $cmd - string  or array of strings
    $attr - hash_ref
      SAVE_TO         - filename to save output
      SKIP_ERROR      - do not finish execution if error on one of commands


  Returns:
    1

get_list($command, $attr)

  Arguments:
    $command
    $attr - hash_ref

  Returns:
    arr_ref

check_access() - checks if mikrotik is accessible

  Returns:
    boolean

upload_key($attr) - Uploads key without password

  Arguments:
    $attr - hash_ref
      ADMIN_NAME    - admin to upload key for
      SYSTEM_ADMIN  - current active admin
      SYSTEM_PASSWD - current password

  Returns:
    1 - if success
