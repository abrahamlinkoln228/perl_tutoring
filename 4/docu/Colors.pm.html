SYNOPSYS

color_hex_to_rgb($hex_color_string)

 Returns:
   arr ref of RGB

rgb_to_hex($rgb_ref) - get hex string from RGB

darken_hex($hex_color_string, $increment_by) - darkens given color

  $hex_color_string : #a825c9  #color
  $increment_by     : coeff to change brightness

  Returns:
    #color or array (r, g, b)

rgb2hls($r, $g, $b)

  $r, $g, $b : 0..255
  Note special name '$bb' to avoid conflict
  with ($a,$b) in sort()

hls2rgb( $h, $l, $s )

  $h     : 0..360 degrees
  $l, $s : 0..1   (inclusive)

_value($n1, $n2, $hue) - helper function for _hls2rgb()

rgb_to_hex2( $r, $g, $b )

 Takes a (r,g,b) triple of numbers (possibly floats) and returns
 - a string like '#36fe25' in scalar context
 - a triple of corresponding integers in array context
